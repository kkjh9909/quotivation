Quotes API 사용법

다음은 Quotes API의 사용법입니다:

  • Base URL: https://api.quotivation.kr/v1/quotes
  • Headers: 없음
  • Parameters(Optional): category
  • Parameters(Optional): page

예제 요청:

curl -X GET "https://api.quotivation.kr/v1/quotes?category=영감&page=1"
